Create data/scrape_flavordb.py as a standalone runnable script. Structure:

1. Imports: os, logging, json, argparse, pandas, tqdm, tqdm.contrib.logging, requests_cache

2. Logging setup at module level:
   - FileHandler("logs/pipeline.log") and StreamHandler()
   - Format: "%(asctime)s %(levelname)-8s %(message)s"
   - Level: INFO

3. Constants:
   - BASE_URL = "https://cosylab.iiitd.edu.in/flavordb/entities_json?id="
   - CACHE_PATH = "data/raw/flavordb_cache"
   - OUT_INGREDIENTS = "data/raw/ingredients.csv"
   - OUT_MOLECULES = "data/raw/molecules.csv"
   - MAX_ID = 1100
   - CONSECUTIVE_404_STOP = 10

4. scrape_flavordb(force=False) function:
   a. Check idempotence: if not force and OUT_INGREDIENTS exists → log SKIP and return early
   b. Create logs/ dir if missing (os.makedirs("logs", exist_ok=True))
   c. Create data/raw/ dir if missing
   d. Validate response on first fetch: if Content-Type is not application/json, raise RuntimeError with clear message (RESEARCH.md pitfall — HTML response means wrong URL)
   e. Build requests_cache.CachedSession with allowable_codes=[200, 404], expire_after=None, cache_name=CACHE_PATH
   f. Iterate entity IDs 1 to MAX_ID:
      - Use tqdm with logging_redirect_tqdm context
      - Track consecutive 404s; break after CONSECUTIVE_404_STOP
      - On 200: parse JSON, append row to ingredients list, extend molecules set
      - On 404: increment failures, log at DEBUG level (not INFO — too noisy)
      - On other status: log WARNING with status code, increment failures
      - On exception: log ERROR, increment failures, continue
   g. ingredients list: each dict has keys: ingredient_id, name, category, molecules_json
      - ingredient_id = entity_id (int)
      - name = data["entity_alias_readable"]
      - category = data["category_readable"]
      - molecules_json = json.dumps(data.get("molecules", [])) — store molecule list as JSON string for later parsing
   h. molecules: use a dict keyed by pubchem_id to deduplicate; each entry: pubchem_id, common_name, flavor_profile
   i. Write DataFrames: pd.DataFrame(ingredients).to_csv(OUT_INGREDIENTS, index=False)
      pd.DataFrame(molecules.values()).to_csv(OUT_MOLECULES, index=False)
   j. Print summary: logger.info(f"FlavorDB: {scraped} scraped, {failed} failed (404)")

5. if __name__ == "__main__": parse --force flag and call scrape_flavordb(force=args.force)
