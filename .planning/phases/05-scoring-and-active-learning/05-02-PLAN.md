    ## compute_all_pairs() — complete implementation
    Follow the pattern from RESEARCH.md Pattern 1 exactly:
    1. Build (n, 128) float32 CPU tensor from embeddings dict.
    2. sim = emb @ emb.T (CPU matmul, ~3.8 MB at n=1000).
    3. rows, cols = torch.triu_indices(n, n, offset=1).
    4. pairing_scores = torch.sigmoid(sim[rows, cols]).numpy() — MUST use sigmoid (per locked decision "dot product must be normalized").
    5. molecular_overlap: Python set Jaccard per pair. Use set intersection/union. If both sets empty, overlap=0.0.
    6. recipe_familiarity: For each (r, c) pair, look up max(co_occurrence.get((names[r],names[c]),0), co_occurrence.get((names[c],names[r]),0)) / max(max_cooc, 1). If co_occurrence is empty, all familiarity=0.
    7. surprise_scores = pairing_scores * (1 - recipe_familiarity) * (1 - molecular_overlap * 0.5).
    8. Build DataFrame with 7 columns. Apply pd.cut() with LABEL_BINS/LABEL_NAMES. Sort by surprise_score descending. reset_index(drop=True). Return.

    def get_top_pairings(ingredient_name: str, n: int = 10) -> list[dict]:

        return df[mask].head(n).to_dict(orient="records")

        return df.nsmallest(n, "_unc").drop(columns="_unc").to_dict(orient="records")

    def run_scoring(force: bool = False) -> Path:
        """
        Compute all ingredient pair surprise scores and save to scoring/scored_pairs.pkl.
        Returns path to scored_pairs.pkl.
        Skips if file already exists unless force=True.

    Implementation inside run_scoring():

    1. Skip-if-exists check:
       if SCORED_PAIRS_PATH.exists() and not force:
           logger.info("scored_pairs.pkl exists — skipping. Pass force=True to recompute.")
           return SCORED_PAIRS_PATH

    2. Validate Phase 4 inputs:
       if not EMBEDDINGS_PATH.exists():
           logger.error(f"Missing {EMBEDDINGS_PATH} — run Phase 4 first")
           sys.exit(1)
       if not GRAPH_PATH.exists():
           logger.error(f"Missing {GRAPH_PATH} — run Phase 3 first")
           sys.exit(1)

    3. Load embeddings:
       with open(EMBEDDINGS_PATH, "rb") as f:
           embeddings = pickle.load(f)  # dict {name: np.ndarray(128,)}
       logger.info(f"Loaded {len(embeddings)} ingredient embeddings")

    4. Build co_occurrence dict and molecule_sets from graph:
       hetero_data = torch.load(GRAPH_PATH, map_location="cpu", weights_only=False)
       Build ingredient_idx_to_name by inverting hetero_data.ingredient_id_to_idx.
       Extract co_occurrence: iterate edge_index of ("ingredient","co_occurs","ingredient")
       and edge_attr; build {(name_a, name_b): weight} dict.
       Extract molecule_sets: iterate edge_index of ("ingredient","contains","molecule")
       — for each ingredient idx, collect the set of molecule idx values.
       If hetero_data does not have ingredient_id_to_idx, fall back to positional mapping
       using list(embeddings.keys()) and log a WARNING.

    5. Compute:
       logger.info("Computing all ingredient pairs (vectorized)…")
       df = compute_all_pairs(embeddings, co_occurrence, molecule_sets)

    6. Save atomically:
       save_scored_pairs(df)

    7. Summary table (MUST match format established in Phase 1):
       n_pairs = len(df)
       label_counts = df["label"].value_counts()
       logger.info(f"Scored {n_pairs:,} pairs.")
       for label in ["Surprising", "Unexpected", "Classic"]:
           count = label_counts.get(label, 0)
           pct = 100 * count / n_pairs if n_pairs > 0 else 0
           logger.info(f"  {label}: {count:,} ({pct:.1f}%)")
       logger.info(f"Saved to {SCORED_PAIRS_PATH}")
       return SCORED_PAIRS_PATH
