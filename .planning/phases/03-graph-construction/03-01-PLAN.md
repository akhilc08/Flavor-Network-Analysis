- Load payload["graph"]; get data["ingredient"].x

- Assert x.dim() == 2 and x.shape[0] >= 500 (num_nodes threshold)

**test_molecule_features** (GRAPH-03)
- Load payload["graph"]; get data["molecule"].x

- Assert x.dim() == 2 and x.shape[0] >= 2000
- Assert x.shape[1] == 1030 (6 descriptors + 1024 Morgan bits)

**test_contains_edges** (GRAPH-04)
- Load payload["graph"]; get edge_index and edge_attr for ('ingredient','contains','molecule')
- Assert edge_index.shape[0] == 2 and edge_index.shape[1] > 0
- Assert edge_index[0].max() < data["ingredient"].num_nodes
- Assert edge_index[1].max() < data["molecule"].num_nodes
- Assert (edge_attr > 0).all() — all weights positive

**test_cooccurs_edges** (GRAPH-05)
- Load payload["graph"]; get co_occurs edge_index and edge_attr
- Assert edge_index.shape[0] == 2 and edge_index.shape[1] > 0
- Assert (edge_attr >= 0).all() and (edge_attr <= 1).all() — normalized weights in [0,1]
- Assert edge_index[0].max() < data["ingredient"].num_nodes
- Assert edge_index[1].max() < data["ingredient"].num_nodes

**test_structural_edges** (GRAPH-06)
- Load payload["graph"]; get structurally_similar edge_index and edge_attr
- Assert edge_index.shape[0] == 2 and edge_index.shape[1] > 0

**test_validation_gate** (GRAPH-07)

- Assert zero overlap in both directions: no (s,d) or (d,s) from test positives appears in train set
