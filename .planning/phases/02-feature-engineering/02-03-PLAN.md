Public function `compute_molecule_features(pubchem_id: int, smiles: str | None, common_name: str) -> dict`:
- Returns full row dict including pubchem_id, smiles, all 6 descriptors, morgan_fp_bytes
- On null or empty smiles: return row with all descriptor/fp columns as None (do NOT raise)
- On Chem.MolFromSmiles returning None: log WARNING with pubchem_id, common_name, smiles[:80]; return null row
- Morgan fingerprint: fp.ToBitString().encode() — stores 1024-byte ASCII string
- Add comment: "# Phase 3 decode: (np.frombuffer(fp_bytes, dtype=np.uint8) == ord('1')).astype(np.float32)"

Public function `compute_tanimoto_edges(fps_and_ids: list[tuple[int, object]], threshold: float = 0.7) -> list[dict]`:
- fps_and_ids: list of (pubchem_id, ExplicitBitVect) for non-null fingerprints only
- Use DataStructs.BulkTanimotoSimilarity(fps[i], fps[i+1:]) per RESEARCH.md pattern
- Filter to similarity > threshold
- Return list of {"mol_a_pubchem_id": int, "mol_b_pubchem_id": int, "similarity": float}
- All-pairs for 1788 molecules completes in ~0.4s — no batching needed

Public function `build_molecule_df(force: bool = False) -> pd.DataFrame`:
- Skip-if-exists: if MOLECULES_PARQUET exists and TANIMOTO_PARQUET exists and not force, log [SKIP] and return pd.read_parquet(MOLECULES_PARQUET)
- Gate check: load pubchem_cache.json; verify set(cache.keys()) == set(str(id) for id in molecules_df.pubchem_id). Raise ValueError if gate fails (do not proceed with partial cache).
- For each molecule in molecules_df (tqdm): call compute_molecule_features; collect rows
- Build fps_and_ids from rows with non-null morgan_fp_bytes (reconstruct ExplicitBitVects from bytes using DataStructs.CreateFromBitString)
- Call compute_tanimoto_edges(fps_and_ids)
- Write molecules.parquet (index=False, engine="pyarrow")
- Write tanimoto_edges.parquet (index=False, engine="pyarrow")
- Log summary: total_molecules={n}, with_descriptors={n}, null_smiles={n}, null_sanitization={n}, tanimoto_edges={n}
- Return molecules DataFrame

IMPORTANT — reconstructing ExplicitBitVect from stored bytes for Tanimoto:
```python
# morgan_fp_bytes is ASCII bit string bytes (e.g., b'010110...')
fp = DataStructs.CreateFromBitString(fp_bytes.decode())
```
This correctly reconstructs the ExplicitBitVect for BulkTanimotoSimilarity.
